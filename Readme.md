Simple Page CV

This project is a simple CV webpage made using HTML and CSS.

Files in this project
Simple-Page-CV
 └── CV
     ├── index.html
     └── style.css

index.html – contains the structure of the CV page

style.css – contains the styling for the page

How to run the project

Download or clone this repository.

Open the CV folder.

Find the file index.html.

Open it in any web browser.

The CV page will open and you can see the design.
