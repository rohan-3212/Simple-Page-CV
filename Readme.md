A simple single-page Curriculum Vitae (CV) built using only HTML.
This project demonstrates how to structure a personal CV webpage using semantic HTML elements without CSS or JavaScript.

📌 Project Overview

The Simple-Page-Cv project focuses on building a clean and well-structured CV page using pure HTML. The goal is to practice HTML fundamentals, including document structure, semantic tags, lists, and metadata.

This project is suitable for beginners who want to strengthen their understanding of HTML layout and structure.

✨ Features

Single page CV layout

Built entirely with HTML5

Proper heading hierarchy

Semantic content structure

Open Graph meta tags for social sharing

Contact email link

Skills, education, and experience sections

External profile links

🛠 Technologies Used

HTML5

No CSS or JavaScript is used in this project.
